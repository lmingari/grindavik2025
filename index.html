Model configuration

  • The FALL3D model was driven using 72-h forecasts from GFS
  • The model is initialized every day at 00:00 UTC using restart
  • Initial condition is taken from the 24-h forecast from the previous day
  • SO2 emission flux was assumed to be 100 kg/s with a Suzuki vertical distribution up to 1500m
